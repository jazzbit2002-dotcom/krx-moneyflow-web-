F29 Pattern Match

이 흐름은 어떤 패턴과 닮았을까요?

가격 흐름을 학습용 패턴과 비교해 닮은 순서로 보여줍니다. 어떤 패턴인지 고르고 판단하는 것은 사용자 몫입니다.

서버에 저장된 종목이면 바로 분석됩니다. 인기 종목 통계에도 반영됩니다.

티커를 함께 입력하면 서버 데이터와 대조해 검증하고, 인기 종목·패턴 개선에 반영됩니다. 티커 없이도 분석은 가능합니다.

입력한 종가 흐름 (교육용 형태 비교)

유사도는 학습용 형태 닮음 점수입니다. 상승·하락 확률이 아닙니다.

닮은 패턴 TOP 3
아직 특정 패턴이 뚜렷하지 않은 흐름입니다. 형태가 더 뚜렷해져야 어느 패턴인지 말할 수 있어, 지금 순위는 참고로만 보세요.
인베스팅닷컴 과거 데이터 가져오기
  1. 인베스팅닷컴 검색창에 종목 티커를 입력합니다. (예: 테슬라 → TSLA)
  2. 종목 페이지에서 일반 → 과거 데이터 탭으로 이동합니다.
  3. 표에서 날짜·종가·시가 등을 원하는 기간까지 드래그해 복사합니다.
  4. 여기 붙여넣기 탭에 티커(예: TSLA)를 쓰고, 복사한 데이터를 아래 칸에 붙여넣습니다.

인베스팅 자료 붙여넣기는 반드시 티커와 함께 입력하세요.
